What is it?

a11yDropDown.js is a jQuery plugin that provides an accessible alternative to custom drop-down menus. Often we want to have a stylish drop-down menu for "Actions" or "Options", this plugin accomplishes that but unlike other drop-down menu plugins a11yDropDown is accessible via mouse & keyboard.

!important

a11yDropDown.js was initially built to run on page load without writing any javascript. To extend the plugin with options, additional data-attributes are set on the element. See the available options below.
a11y is an abbreviated numeronym for Accessibility, where the number 11 refers to the number of letters omitted.

Options

Data Attribute Options

Attribute Values Description
data-position String
"left"
"right"		 Position the drop-down menu to either side of the drop-down link. Default position is "right".
data-offset-pixel Number Offset the drop-down menu by 'x' number of pixels. Can be set to a negative number to move left or right.
data-offset-element String
Element ID
data-autohide String
"true"
"false"		 By default the menu auto-hides, but this feature disables that and closing the menu occurs when you click anywhere else or tab out.
